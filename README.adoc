`h.h` is constructed from a series of components with `m4`.

[source,sh]
----
m4 h.h.m4 > h.h
----

If you only need a subset of the library's functionality, you can omit the rest.
Components mirror the filenames in `/include`.

[source,sh]
----
m4 -Dcomponents="core args" h.h.m4 > h.h
----

Alternatively, just use the individual headers.
